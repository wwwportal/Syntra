Overview

kalima-db is a SQLite database containing the Quranic text with full morphological annotation. It serves as the foundation for all analysis performed by kalima-mcp and kalima-math.

Location: HuggingFace: Syntra-Env/kalima-db

The database is automatically downloaded on first use by kalima-mcp, or can be manually downloaded using the HuggingFace Hub API.

Data Structure

The database contains:

  • Text content: The Quranic text (Arabic and normalized)
  • Morphological features: Root, POS, verb form, aspect, person, number, gender, case
  • Linguistic annotations: Word types, word instances, morpheme types
  • Reference data: Traditional tafsir translations (Sahih International)

Storage & Distribution

The database is:

  • Stored as kalima.db on HuggingFace Datasets
  • Downloaded via huggingface_hub on first run
  • Cached locally at ~/.scholar/data/kalima.db

This approach keeps the repository lightweight (no large binary files in git) while ensuring the data is versioned and accessible.

Access

Access via kalima-mcp tools, or directly through the database layer in kalima-mcp:

from src.db import get_connection
conn = get_connection()

Manual download:

from huggingface_hub import hf_hub_download
db_path = hf_hub_download(repo_id="Syntra-Env/kalima-db", filename="kalima.db", repo_type="dataset")
