# Geometer

Geometer provides the mathematical foundations for Syntra research, implementing formal systems for analyzing classical Arabic text.

## Overview

Geometer implements:

- **HUFD (Holonomic Unified Field Dynamics)**: Models text as an su(2) gauge field
- **UOR (Universal Object Reference)**: Content addressing system using SHA-256
- **Mathematical Utilities**: Core math layer for semantic analysis

## Key Concepts

### Curvature ($R_{\mu\nu}$)

Measures semantic rotation of words across contexts.

### Holonomy ($\oint H_\mu$)

Tests for semantic constants—roots with invariant meaning across the corpus.

### Phase-Lock ($\Phi$)

Coupling metric between verses for boundary detection.
