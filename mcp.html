Overview

kalima-mcp is a Model Context Protocol (MCP) server that provides tools for Quranic analysis. It connects to kalima-db for data and uses kalima-math for computations.

Repository: GitHub: Syntra-Env/kalima-mcp

The MCP protocol allows any AI assistant (Claude, Cursor, etc.) to use these tools directly.

Tools

Available tools include:

ToolDescription
get_verse_with_contextGet verse text with word morphemes and root analysis
compare_with_traditionalCompare analysis with traditional tafsir (Sahih International)
detect_boundariesDetect speaker/topic boundaries in a passage
analyze_verse_emphasisAnalyze which features carry emphasis in a verse
decompose_addressDecompose a UOR address into components
locate_identityFind entities by their UOR address

Architecture

The MCP server:

  1. Receives requests via stdio (MCP protocol)
  2. Queries kalima-db for data
  3. Passes data through kalima-math functions
  4. Returns results to the client

└──────────────┘            │                  │

                      ┌─────────────┐          │

                      └─────────────┘

Running

Run locally:

pip install scholar-mcp-server
scholar

Or via Docker:

docker run ghcr.io/syntra-env/kalima-mcp:latest
