System Overview

The kalima system consists of four main projects:

  • kalima-math — Mathematical foundations (pure math, no dependencies)
  • kalima-db — Quranic dataset (stored on HuggingFace)
  • kalima-mcp — MCP server and tools (uses math + db)
  • kalima-ml — ML experiments (in development)

Data Flow

Analysis flow:

  1. User asks AI assistant for analysis
  2. AI uses kalima-mcp tools via MCP protocol
  3. kalima-mcp queries kalima-db for Quranic data
  4. Data is processed through kalima-math functions
  5. Results returned to user

ML flow (future):

  1. Model loads from HuggingFace (syntra/kalm-model)
  2. Can use kalima-math coordinates as features
  3. Can use kalima-db for training data
  4. Results exposed through kalima-mcp
